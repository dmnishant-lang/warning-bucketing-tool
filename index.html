How to use this tool each cycle

  1. Export the main fraud detection table and all three warning tables as CSVs from your dashboard
  2. Drop each file into its zone above (or click to browse), then hit Run bucketing
  3. Review each bucket — only shifts from this month and last month are included
  4. Use Download CSV or Show CSV → Copy all to get the output into your email service
  5. After sending emails, move workers up into the next warning tier in your source so history stays accurate for next run
